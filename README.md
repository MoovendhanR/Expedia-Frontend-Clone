## Run Locally

Clone the project

```bash
  git clone https://github.com/Omkar-WD/Expedia-Frontend-Clone.git
```

Go to the project directory

```bash
  cd Expedia-Frontend-Clone
```

Install dependencies

```bash
  npm install react-router-dom
  npm install axios
```

Start the server

```bash
  npm run start
```
